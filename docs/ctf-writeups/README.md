# CTF Writeups

## Overview

This directory structure contains writeups for CTF (Capture the Flag) challenges and cybersecurity exercises.

Each writeup documents the analysis, tools, and solution steps used to solve a challenge, preserving the context of where and how techniques were first encountered.

## Structure

| Directory | Platform | Description |
|---|---|---|
| 🧪 [`ctfd/`](./ctfd/README.md) | CTFd | Annual cybersecurity CTF challenges run internally at work |
| 🎄 [`holiday-hack-challenge/`](./holiday-hack-challenge/README.md) | SANS | SANS Holiday Hack Challenge — free annual global cybersecurity event |
| 🧩 [`immersive-labs/`](./immersive-labs/README.md) | Immersive Labs | Halloween-inspired CTF challenges and training lab exercises |

## Relationship to `ctf-techniques/`

Writeups in this directory structure feed the [`ctf-techniques/`](../ctf-techniques/README.md) reference documentation. When a challenge introduces a technique that is worth keeping as a standalone reference (e.g., a new tool, attack pattern, or protocol) it gets extracted and documented there in a reusable, challenge-agnostic form.

The writeups preserve the original challenge context; the techniques documentation distills the reusable knowledge.
