# SANS Holiday Hack Challenge

## Overview

The [SANS Holiday Hack Challenge](https://www.sans.org/mlp/holiday-hack-challenge/) is a free, global cybersecurity game and training event hosted by the SANS Institute.

Run annually each December, it combines a whimsical holiday storyline with hands-on, real-world security challenges designed for players of all skill levels.

Challenges span web exploitation, reverse engineering, hardware hacking, cryptography, forensics, and more; all wrapped in a narrative that unfolds across acts.
