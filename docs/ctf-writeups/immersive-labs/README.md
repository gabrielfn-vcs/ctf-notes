# Immersive Labs

## Overview

[Immersive Labs](https://www.immersivelabs.com/) is a cybersecurity workforce development platform that provides hands-on labs and skill-building exercises.

This directory contains writeups for a Halloween-inspired CTF challenge series, as well as a selection of standalone training lab exercises completed through the platform.
