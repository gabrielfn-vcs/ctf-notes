## Overview

Decipher QR codes to get secret messages and discover the secret word.

## Analysis

* **QR Code 1:** `Creeping through the code, under the moon's glow,`
* **QR Code 2:** `On this eerie Halloween night, the cyber winds blow,`
* **QR Code 3:** `Riddles and mysteries, in the Labs they stow.`
* **QR Code 4:** `Whispering secrets only Immersive Labs know.`

## Solution

The first letter of each message is `C`, `O`, `R`, `W`.

By reordering them, we can find the secret word: `CROW`.
