## 📖 Overview

A structured personal collection of CTF (Capture the Flag) techniques and challenge writeups, built for learning and quick reference.

Writeups are organized by event and platform, preserving challenge context. Techniques are organized by topic — independent of any specific challenge — so they can be reused as a reference across future competitions and labs.

---

## ⚠️ Disclaimer

This repository is for **educational purposes only**. All content documents methods encountered in authorized CTF competitions, sanctioned security training platforms, and legal lab environments. This including all techniques, scripts, and writeups.

Nothing in this repository should be used against systems, networks, or applications without explicit authorization from the owner. Unauthorized use of these techniques may violate local, national, or international law.

## 📂 Repository Structure

| Directory | Description |
|---|---|
| [`ctf-techniques/`](./ctf-techniques/README.md) | Reusable technique reference organized by topic: web, crypto, forensics, network, hardware, and more |
| [`ctf-writeups/`](./ctf-writeups/README.md) | Challenge writeups organized by platform and event, with links back to relevant techniques |

The two directories are designed to complement each other, i.e., writeups link to techniques used, and techniques link back to real-world examples in the writeups.
